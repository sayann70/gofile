# GoFile Uploader
A Simple Script to upload Files to https://gofile.io via Terminal (CLI). Written in Bash.

```bash
curl -O https://raw.githubusercontent.com/sayann70/gofile/refs/heads/main/gofile.sh
```
```bash
chmod +x gofile.sh
```
```bash
./gofile.sh <path/to/file>
```
